# <strong>How to install ❓ </strong></h2>


클라이언트와 서버에서 각각 다음 작업 수행

#### ⚙️Server
```
.env 파일 생성(/server/env.txt 참고)
npm install
npm run dev
```
#### ⚙️Client
```
.env 파일 생성(/client/env.txt 참고)
npm install
npm start
```
